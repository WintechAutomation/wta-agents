## 주요 발견사항

### 정상 동작
- 통계 카드: 총 견적 248건, 총금액, 고객사 수, 연도 범위 정상 표시
- 연도별 필터 (2021~2026) + 고객사별 필터 (한국야금, 교세라 등) 동작
- 검색창 ("CS번호, 고객명, 제목 검색") — 실시간 필터링 + 검색 버튼
- 모바일: 하단 탭 네비게이션, 통계 카드 2열 레이아웃 정상

### 미완성 검증 (5, 6번)
**원인**: 편집/삭제 아이콘이 SVG 버튼으로 구현됨. aria-label/title 속성이 없어 Playwright 선택자로 특정 불가.
- 편집 아이콘(연필) · 삭제 아이콘(휴지통) UI에는 표시 확인 (`screenshots/02-list-loaded.png` 참조)
- 모달 오픈 자동화를 위해 명확한 aria-label 속성 추가 권장

### viewer 권한 관련
- 편집/삭제 아이콘이 viewer 역할에서도 표시됨
- 권한에 따른 버튼 숨김/비활성화 여부는 실제 클릭 후 동작으로 검증 필요
- **권장**: 편집 버튼에 `aria-label="견적 수정"`, 삭제 버튼에 `aria-label="견적 삭제"` 추가 요청 (dev-agent)
