AGV_ELEVATOR_ORG_ULO // 취출 원점위치 원점 위치 잘 맞을 경우 0 SCAN_START_LO // 공급 스캔 시작점. (고객사 사양 플레이트 높이x최대 적재 개수 = 최대 적재 높이) -> 최대 적재 높이보다 적당히 높은 위치로 설정 (10~20정도) SCAN_START_ULO // 취출 스캔 시작점 이하 동일 SCAN_END_LO // 공급 스캔 종료지점 공급 최하단 플레이트 가장 아래의 위치 → 보통 공급 원점 위치와 동일 SCAN_END_ULO // 취출 스캔 종료지점 취출 최하단 플레이트 가장 아래의 위치 → 보통 취출 원점 위치와 동일 AGV_WORKPOS_LO // 플레이트 자동 교체 로봇 공급 시 위치 AGV_WORKPOS_ULO // 플레이트 자동 교체 로봇 취출 시 위치 CLAMP_OFFSET_LO // 공급 클램프 오프셋 공급할 플레이트를 클램프 하기 전 이동할 거리 CLAMP_OFFSET_ULO // 취출 클램프 오프셋 취출할 플레이트를 클램프 하기 전 이동할 거리 UPHOLD_OFFSET_ULO // 하단 센서 위치 보정 오프셋 플레이트 감지 센서 위치 조절이 불가 할 때 플레이트 최하단까지의 남은 거리 좌측 이미지처럼 센서가 감지 중인 위치가 플레이트의 최하단까지 가지 못할 때 필요로 한다. (중앙이나 상단을 보고 있을 때) 별도의 조작 없이 센서가 플레이트 최하단까지 감지 가능한 경우 0 으로 설정한다. ex)센서가 감지중인 위치가 플레이트 최하단까지 5mm 부족할 때 MIN_SIZE // 사용할 플레이트의 최소 두께 MAX_SIZE // 사용할 플레이트의 최대 두께 ACCEPTABLE_OCHA // 플레이트 상단 허용 오차 스캔 후 클램프 위치로 엘리베이터 이동할 때 해당 오차 허용 값 만큼 추가 이동 🔗 원본: https://iwta.atlassian.net/wiki/spaces/~63d21a670d930a766df1049f/pages/8435892472
