(C/B 방향 Tool 학습) [이미지] 이후 C/B 방향구분 버튼을 누르면 방향이 맞다면 넘어가고 방향이 맞지 않다면 맞는 방향으로 회전 시킨다. 방향이 안맞는 경우 [이미지들] 방향이 맞는 경우 [이미지들] 앞/뒷면 구분 현재 인서트가 앞면인지 뒷면인지 확인한다. 인서트 앞면과 뒷면을 확인 후 앞면과 뒷면을 판단할 수 있는 특징을 학습한다. (앞/뒷면 Tool 학습) [이미지] 이후 앞/뒷면 구분 버튼을 누르면 현재 인서트가 앞면인지 뒷면인지 판단한다. 앞면의 경우 [이미지들] 뒷면의 경우 [이미지들] C/B 혼입 비교 인서트의 C/B 모양이 원하는 모양인지 확인한다. 혼입되는 C/B 모양 판단 할 수 있는 특징을 학습한다. (C/B 혼입 Tool 학습) [이미지] 정상인 경우 [이미지] 비정상인 경우 [이미지] NoseR 측정 인서트의 NoseR은 처음 인서트를 잡을때 교점을 이용해 버튼을 누르지 않아도 자동 학습된다. [이미지] NoseR의 영역 테이블을 수정하면 원하는 범위를 설정할 수 있다. (아직 코드에서 수정해야 하지만 추후 UI에서도 수정 가능하도록 추가 예정) [이미지] 이후 NoseR 측정을 누르면 현재 측정중인 곳에 빨간색 점이 생기고 해당 코너의 NoseR을 계산한다. [이미지들] 여러 인서트 종류 확인 완료 🔗 원본: https://iwta.atlassian.net/wiki/spaces/VSNEDU/pages/8743878914
